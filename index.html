Turkish Deasciifier

English: If you don't have a Turkish keyboard but want to write using proper Turkish letters, you can use this system. Write inside the box below and hit Deasciify / Türkçeleştir.

Türkçe: Türkçe klavyeniz yoksa ve Türk diline özgü harfleri kullanmadan yazmak zorunda kalıyorsanız bu sistemden faydalanabilirsiniz. Aşağıdaki kutu içine yazınızı yazın ve Deasciify / Türkçeleştir düğmesine basın.

This converter is the Python implementation of Deniz Yüret's converter (in Emacs Lisp) by Emre Sevinç. If you need something more usable, e.g. in your browser menu, you can use the Firefox add-on or the Google Chrome extension.

Bu sistem Deniz Yüret'in Emacs Lisp programlama dili ile yazdığı sistemin Emre Sevinç tarafından Python dili ile uygulanmış halidir. Bu sistemin benzerini daha pratik olarak kullanmak isterseniz Firefox add-on halini da Google Chrome extension halini web tarayıcınıza kurabilirsiniz.
